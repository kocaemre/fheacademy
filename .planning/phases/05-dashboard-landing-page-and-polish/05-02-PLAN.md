Component:

- AccordionTrigger: Bot icon (lucide-react) + "AI Grader" title + brief subtitle "Get AI-powered feedback on your code"
- AccordionContent contains the grader flow:
  1. **Textarea** for pasting Solidity code — labeled "Paste Your Code", placeholder text, min-height ~200px, font-mono
  2. **"Generate Grading Prompt" button** — disabled when textarea is empty
  3. **Readonly textarea** showing generated prompt — hidden until generated, font-mono, min-height ~300px
  4. **"Copy to Clipboard" button** — uses `navigator.clipboard.writeText()`, shows Check icon for 2 seconds after copy (useState for copied state with setTimeout)

Prompt builder function `buildPrompt(title, criteria, code)`:
- Returns a structured prompt:
  ```
  You are grading a student's FHEVM homework submission.

  ## Assignment: ${title}

  ## Grading Rubric
  ${criteria formatted as bullet list with all fields}

  ## Student's Code
  \`\`\`solidity
  ${code}
  \`\`\`

  ## Instructions
  Grade the student's code against each rubric criterion. For each:
  1. State PASS or FAIL
  2. Explain what was done well or what's missing
  3. Assign a score within the weight range

  End with overall score and summary.
  ```

- Add a note below the component: "This prompt works with any AI model — paste it into ChatGPT, Claude, or any other AI assistant."
